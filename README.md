# Tennis Biomechanics Interactive Calculator

Sinh Cơ Học Tennis Tương Tác - Khám phá 10 công thức cốt lõi chi phối hiệu suất quần vợt

## About

This is an interactive web application that provides 10 core biomechanical formulas for tennis, allowing users to understand and calculate:

1. **Chỉ Số Truyền Tải Quán Tính** - Inertial Transmission Index
2. **Năng Lượng Đàn Hồi** - Elastic Energy
3. **Mô-men Xoắn (Torque)** - Torque calculations
4. **Độ Nặng Của Bóng** - Ball Heaviness Index
5. **Mô-men Quán Tính Vợt (Swing Weight)** - Racket Moment of Inertia
6. **Phân Bổ Trọng Lượng Vợt** - Racket Weight Distribution
7. **Độ Cứng & Hiệu Suất** - Stiffness and Efficiency
8. **Độ Vững Chắc Cổ Tay** - Wrist Stability
9. **Hiệu Suất Di Chuyển** - Movement Efficiency
10. **Bảo Toàn Mô-men Động Lượng** - Conservation of Angular Momentum
